# DOF Document Management System

## Synopsis
Document management system backend API built with Express.js and Prisma.

## Description
This backend service provides RESTful APIs for user authentication, document management, and workflow tracking.

## Environment Variables
- `DATABASE_URL` - PostgreSQL connection string
- `JWT_SECRET` - Secret for JWT token signing
- `PORT` - Server port (default: 5000)

## Install
```sh
npm install
```

## Run
```sh
npm run dev
```
